Interface Description

Game Carousel

The Game Carousel allows you to select the current game. It will dislpay a maximum of five games at a time. The arrows can be used to scroll left or right to change the currently displayed games. The 'Add Game' button allows the user to add a new game to the carousel

Game Information

Game Information displays the cover, title, genre, develovper, total playtime, and number of goals a game has.

Add Game

Add Game allows the user to add a new game. It is activated through the 'Add Game' button in the Game Carousel. It replaces the Game Information until the new game is submitted or the action is canceled.

Goals

Goals allows the user to track goals for their game. The checkmark can be clicked to mark a goal as completed. New goals can be added when the user is adding a new entry to the game.

My Entries

My Entries shows all of the entries made for a game. The selector on the left allows the user to select an entry to be displayed on the right. The selector displays entries by the date they were submitted. The entry shows the time played for the session, the rating of the session and things the user liked or disliked about the session. The circle next to the rating changes color based on the rating starting at red for 1 and changing to green at 10. The 'Edit Entry' button allows the user to edit the current entry.

Add Entry

Submit a New Entry allows the user to create a new entry for the currently selected game. The can log things they liked or disliked, a general rating of 1-10, and the playtime of the session. They can also add a new goal to their goals tracker.

Edit Entry

Edit an Entry allows the user to edit a specified entry. It is accessed through the 'Edit Entry' button in My Entries.

Implementation

This was created using base Svelte. The default demo data is saved in a store. This store is used for further data added by the user.
